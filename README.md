# Fun Fact
`llms.txt` is a plain-text file you put at the root of your website (e.g. `<website>/llms.txt`).

It is a proposed standard that acts as a curated "cheat sheet" or map to guide AI models and agents to a website's most important and clean content. It's a simple, emerging convention just like `robots.txt`, but aimed at AI instead of search crawlers.
